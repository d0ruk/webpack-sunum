Webpack nedir?

Terminalden komutlar vererek konfigüre edebildiğiniz, JS ile yazılmış bir uygulamadır.

> webpack --hide-modules

Ancak, terminale uzun bir komut girmek yerine, CLI'a bir config objesi (node API) vereceğiz.

// webpack.config.js
module.exports = () => {
  return {
    entry: // modül ağının başlama noktası
    output: // nereye ve nasıl çıktı alınacak
    module: // karşılaşılan dosyalara nasıl müdahale edilecek
    plugins: // build prosesinize eklentiler
  }
}
Gereken minimal config entry ve output

Girdi: kaynak kodunuz (modüller ağı) ve konfigürasyonunuz

Çıktı: tarayıcılarda kullanılabilecek bundle dosyaları

Modül yapısı

// a.js
module.exports = 42;
// b.js
const a = require("./a.js");
console.log(a); // 42
