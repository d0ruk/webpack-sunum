Webpack nedir?

Girdi: kaynak kodunuz ve konfigürasyonunuz

Çıktı: tarayıcılarda kullanılabilecek bundle dosyaları

CLI

Terminalden komutlar vererek konfigüre edebildiğiniz, JS ile yazılmış bir uygulamadır.

> webpack --hide-modules

Ancak, terminale uzun bir komut girmek yerine, CLI'a bir config objesi vereceğiz.

// webpack.config.js
module.exports = () => {
  return {
    entry: // girdi ayarları
    output: // çıktı ayarları
    module: // karşılaşılan dosyalara nasıl müdahale edeceğiniz
    plugins: // build prosesinize eklentiler
  }
}
Gereken minimal config entry ve output
